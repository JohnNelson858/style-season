Outfitter for All Seasons!

This page uses JavaScript to change the look and feel of the Eddie Browser page when a season is clicked. The script reads the season from the link, changes the page background color, updates the seasonal message, changes the logo image, and changes the clothing image. Clicking Default restores the original page settings.

The colors of the seasonal images are as follows:

  • Spring: #2B7129
  • Summer: #EBA52B
  • Fall: #A81124
  • Winter: #005393
  • Default: gray

Eddie Browser is a seasonal wear outfitter that changes with the weather. Choose a season above to see the page update with a new color scheme, logo, clothing image, and seasonal message.
